# Contributing to SecOC Simulator

Thanks for your interest in contributing! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/secoc-simulator.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Install dependencies: `pip install -r requirements.txt`
5. Make your changes
6. Test: `python -m secoc_simulator` and `python run_dashboard.py`
7. Commit and push
8. Open a Pull Request

## What to Contribute

- Bug fixes
- New attack scenarios
- Additional MAC algorithms
- CAN FD improvements
- Documentation improvements
- Test coverage
- Integration with real CAN interfaces (python-can)

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings to public methods
- Keep functions focused and small

## Reporting Issues

Open an issue with:
- Steps to reproduce
- Expected vs actual behavior
- Python version and OS
